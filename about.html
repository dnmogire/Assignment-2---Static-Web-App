About This Project

Project Overview

This Football Players Gallery is a static web application created as part of the CSC 340 course assignment. The project demonstrates fundamental web development skills including HTML, CSS, and JavaScript while showcasing information about various football players from around the world.

Purpose

The main purpose of this website is to provide an interactive gallery where users can browse through different football players, learn about their careers, and filter players based on various criteria such as name, position, nationality, or club. The site features a clean, responsive design that works well on both desktop and mobile devices.

Features

This website includes several key features:

  • Home page with a gallery of 5 football players
  • Search functionality to filter players in real-time
  • Detailed player profile page with statistics and career information
  • About page explaining the project (this page)
  • Form page to add new players to the database
  • Responsive design that adapts to different screen sizes
  • Clean and simple navigation throughout the site

Technologies Used

This project was built using standard web technologies:

  • HTML5: For structuring the content and pages
  • CSS3: For styling and responsive design
  • JavaScript: For dynamic content and search functionality
  • GitHub Pages: For hosting the static website

Development Process

The website was developed following web development best practices. The project started with planning the structure and layout, then building the HTML pages, styling them with CSS, and adding interactivity with JavaScript. The code was version-controlled using Git and deployed to GitHub Pages for public access.

Future Improvements

Potential future enhancements could include adding more players to the gallery, implementing sorting options, creating individual detail pages for each player, adding team comparisons, and integrating with a real database to dynamically manage player information.
